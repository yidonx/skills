# 查询中国城市天气

## 触发条件
用户询问中国城市天气（铁岭、沈阳、杭州等）

## 方式一：命令行（备选，已知局限）

使用 `wttr.in` 服务：

```bash
curl -s "wttr.in/城市名?format=j1"
```

⚠️ **已知局限**：在 Hermes 环境中，curl 可能被安全扫描拦截（`Schemeless URL in sink context`），导致请求被拒绝。此时直接改用方式二。

**输出格式说明**
- `current_condition` → 当前天气（温度、体感、湿度、风速、风向）
- `weather` 数组 → 每天预报（含 `maxtempC`/`mintempC`、`hourly` 每3小时细分）
- `avgtempC` → 日均温

**返回内容示例**
```json
{
  "current_condition": [{ "temp_C": "21", "humidity": "88", "windspeedKmph": "18" }],
  "weather": [{ "date": "2026-05-27", "maxtempC": "28", "mintempC": "17", "hourly": [...] }]
}
```

**优点**
- 极速（无需启动浏览器）
- 国内可访问
- 返回 JSON，可编程解析

---

## 方式二：weather.com.cn 搜索框（推荐，通用性最强）

**适用场景**：curl/wttr.in 被拦截、或城市代码未知、或不确定代码是否正确。

1. 打开 `https://www.weather.com.cn/`
2. 在顶部搜索框输入城市名（如"菏泽"）
3. 按 `Enter` 跳转到该城市天气页面
4. 点击"7天"标签读取完整预报

此方式不依赖城市代码表，对任意城市均有效，是最可靠的浏览器方案。

---

## 方式三：直接 URL（已知代码时）

当确认城市代码时，可直接访问：
`https://www.weather.com.cn/weather/城市代码.shtml`

| 城市 | 代码 |
|------|------|
| 北京 | 101010100 |
| 上海 | 101020100 |
| 杭州 | 101210101 |
| 深圳 | 101280601 |
| 广州 | 101280101 |
| 成都 | 101270101 |
| 武汉 | 101200101 |
| 南京 | 101190101 |
| 沈阳 | 101070101 |
| 铁岭 | 101071101 |

> ⚠️ 此表不完整，中国天气网支持数百个城市。对于表中没有的城市，**必须使用搜索框方式**（方式二），不要尝试猜测代码。

---

## 验证方式
- 命令行：curl 成功返回 JSON 且含 `temp_C` 字段
- 浏览器：页面成功加载并显示天气预报数据
