## 游戏基础规则

### 对局结构

- 每局 8 名玩家，最后存活者获胜
- 每回合分为：**准备阶段**（买棋子/调阵容）→ **战斗阶段**（自动战斗）
- 输掉战斗会扣血，扣完 0 血淘汰
- 前期输血量少（1~2点），后期越来越多

### 金币与经济

| 金币收入来源 | 说明 |
|------------|------|
| 基础收入 | 每回合固定 5 金币 |
| 利息 | 每持有 10 金币获得 1 利息，上限 5 金币/回合 |
| 连胜/连败奖励 | 连续胜/败可额外获得 1~3 金币 |
| 过路费 | 特定回合系统奖励 |

**利息最大化原则**：尽量保持 50 金币以上，每回合稳拿 5 利息，经济滚雪球。

### 等级与升级

| 等级 | 每费棋子出现概率（近似） |
|------|------------------------|
| 4级 | 1费55% / 2费30% / 3费15% |
| 5级 | 1费45% / 2费33% / 3费20% / 4费2% |
| 6级 | 1费25% / 2费35% / 3费30% / 4费8% / 5费2% |
| 7级 | 1费19% / 2费30% / 3费35% / 4费12% / 5费4% |
| 8级 | 1费15% / 2费20% / 3费35% / 4费20% / 5费10% |
| 9级 | 1费10% / 2费15% / 3费30% / 4费25% / 5费20% |

> 概率为通用参考，具体数值以当前版本为准。

## 牌池机制

### 牌池总量（每种棋子）

| 费用 | 每种棋子数量 |
|------|------------|
| 1费  | 22张 |
| 2费  | 20张 |
| 3费  | 17张 |
| 4费  | 10张 |
| 5费  | 9张  |

### 关键规则

- 牌池**全场8人共享**，你买走的牌别人就看不到
- 卖出棋子会**回到牌池**，不会永久消失
- **合成三星**（消耗9张）是牌从池子里永久减少的唯一方式
- 看到某张牌长时间不出现 → 大概率有人在抢同一张

### 三星所需张数

```
1星（1张）→ 2星（3张合1）→ 3星（3个2星 = 9张合1）
共需 9 张同名同费棋子
```

## 三星策略

### 按费用分类

| 费用 | 建议等级 | 策略要点 |
|------|---------|---------|
| 1费  | 卡 4~5 级 | 概率最高，早期强行三星；注意抢池 |
| 2费  | 卡 5~6 级 | 中前期核心，观察他人是否抢同张 |
| 3费  | 6~7 级刷新 | 均衡点，适合中期过渡核心 |
| 4费  | 升 8 级后刷 | 池子小（10张），全场基本只能1人三星 |
| 5费  | 通常不追三星 | 池子仅9张，成本极高，看运气 |

### 三星决策流程

1. 确认目标棋子费用
2. **控制等级**：不盲目升级，保住高概率出牌区间
3. **观察牌池**：商店多久没出现目标牌？
4. **看小地图**：有人和你买同一张棋子吗？
5. **判断继续/放弃**：池子被抢光则考虑换思路

## 经济运营节奏

### 标准运营模板

```
1-3回合   滚存金币，不乱买，奔向50金利息
4-6回合   视情况选择：强势开局推血 / 继续存钱
7-10回合  根据当前生命值决定是否开始刷新
中后期    50金以上稳定后开始大量刷新/升级
```

### 两种主要路线

**存钱路线（利息流）**
- 保持 50 金以上，每回合 5 利息
- 适合场面不弱、能扛住前期伤害的情况
- 后期经济爆发，大量刷新找核心棋子

**快攻路线（强推流）**
- 不存钱，每回合花完金币买最强棋子
- 适合开局拿到强力羁绊/装备
- 靠连胜奖励补充经济

## 羁绊机制

- 场上凑齐同一**职业或种族**的棋子达到激活数量，触发羁绊效果
- 同一棋子可同时贡献多个羁绊（比如一个"刺客+暗影"单位同时计入两个羁绊）
- 三星棋子仍只算**1个羁绊数**
- 海克斯强化可能额外提供羁绊数量加成

## 装备系统

### 合成原则

- 基础装备 + 基础装备 = 合成装备（共9种基础 × 组合方式）
- 装备只能装在**上场棋子**身上，备战席棋子无法穿装
- 同一合成装备可叠加（部分有上限）

### 装备优先级思路

1. 先确认核心输出棋子是谁
2. 根据棋子类型（物理/魔法/坦克）匹配装备方向
3. 前期可先用基础装备过渡，凑齐后合成
4. 不要为了凑装备强行改变阵容方向

## 对局决策框架

### 每回合检查清单

```
[ ] 当前金币是否到50？利息是否最大化？
[ ] 生命值是否安全？需要补强还是继续存钱？
[ ] 当前阵容核心是否成型？
[ ] 目标三星棋子牌池状况如何？
[ ] 是否需要升级提高高费出现率？
[ ] 装备是否合理分配？
```

### 灵活应变原则

- **不要固执于开局计划**：根据商店刷出的棋子和装备随机应变
- **观察对手阵容**：针对性放置棋子位置（前排/后排）
- **血量管理**：连败也是一种策略（连败奖励 + 省钱）

## 常见问题解答

**Q：刷新商店会消耗什么？**
A：每次刷新固定消耗 2 金币，不会影响利息计算基数。

**Q：被淘汰的玩家的棋子去哪了？**
A：全部回到牌池，对剩余玩家可见。

**Q：同一张棋子被合成三星后，9张永久消失吗？**
A：是的，合成三星是牌从池子永久减少的唯一方式，卖掉三星也只回3张2星（即9张）。

**Q：备战席上的棋子有什么用？**
A：可以暂存棋子凑星级，但不参与战斗、不提供羁绊数量。

**Q：如何判断应该抢哪个阵容？**
A：优先考虑：① 开局装备方向；② 早期刷到的棋子；③ 当前版本强势阵容；④ 避开他人正在走的路线。
